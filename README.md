# Digital Element test

## Описание

### `Node v20.9.0 и выше` <br>

### `npm 10.1.0 и выше` <br>

## Установка

**Клонируйте репозиторий:**

```bash
git clone https://github.com/PavelPotapov/testDE.git
cd testDE
```

**Установите зависимости:**

```bash
npm install
```

## Запуск

```bash
npm run dev
```
